# PRD 与占位框架对齐说明

**事实来源**：以 [docs/PRD.md](PRD.md) 为准。源 Word：`竞拍之王.docx`、`竞拍之王 - 数值简化.docx`。

## 占位框架摘要（对话计划）

| 维度 | 占位要点 |
|------|-----------|
| 主题 | Bid：出价与风险；King：权柄/席位/可轮换优势 |
| 核心循环 | 准备 → 竞拍 → 结算与权柄转移 → 至终局 |
| 建议写清 | 胜利条件、货币与破产、竞拍信息公开度、王权与反滚雪球、人数与局时 |

## 当前对齐状态（v0.1 PRD 整合稿后）

| 占位项 | PRD 覆盖情况 | 备注 |
|--------|--------------|------|
| 一句话与愿景 | **已覆盖** | PRD §1 |
| 核心循环 | **已覆盖** | PRD §4（三阶段 + 三轮 + 结算）；与占位「准备→竞拍→权柄」同构，权柄体现为「最高者得额外信息」 |
| 胜利条件与平局 | **部分覆盖** | PRD §7 为原则级；未写平局细则 |
| 经济 / 货币 / 破产 | **部分覆盖** | 门票、卖出、场次倍率有；破产/出局未定义 |
| 竞拍规则（含信息结构） | **已覆盖** | PRD §4、§6、§8；明确不做暗标主流程、中场公开最高价 |
| 王权 / 权柄与反滚雪球 | **刻意偏离后已解释** | 无「国王席位」叙事；**轮内优势** = 出价最高者得**额外信息**；反滚雪球细节仍待策划（见 PRD §12） |
| 目标人数与局时 | **已覆盖** | 4 人、40s/回合、3 轮、约 60s+/轮含结算 |
| 边界与非目标 | **已覆盖** | PRD §10 |
| 术语表 | **已覆盖** | PRD §11 |

**冲突项**：与初版占位「King=王冠叙事」不完全一致——PRD 采用「鉴宝竞拍 + 信息权」；属**主题映射**，非文档矛盾。

**仍待 PRD/策划补全的缺口**：见 [docs/PRD.md §12](PRD.md)（门票数值、档位金额、流拍规则、额外信息边界、w1/w2/w3 字段映射、Word 内嵌表导出等）。
